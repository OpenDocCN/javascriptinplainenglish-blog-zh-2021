理解地图🗺

当我听到“地图”这个词时，我首先想到的是世界地图😆。我知道这是一个糟糕的笑话，但我不得不这样做。不管怎样，map是一个JavaScript数组方法，这意味着这个方法只能在数组上使用。考虑到arr 是一个包含元素[1,2,3,4,5]的数组，现在 VS代码的代码完成工具提供了一个方便的下拉菜单，当我键入 map 时，意味着这个变量 arr 有这个叫做 map 的方法可用。
