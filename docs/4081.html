设置和安装

我们使用最新的 React navigation 6 在屏幕和 NativeBase 组件之间切换来制作UI。要使用它，我们必须进行一些安装。所以，让我们安装它。确保您已经安装了高于或等于41的expo版本。

为了导航

  • npm install @react-navigation/native
  • npm install react-native-screens react-native-safe-area-context
  • npm install @react-navigation/stack react-native-gesture-handler

对于NativeBase

  • npm install native-base styled-components styled-system
  • expo install react-native-svg
  • expo install react-native-safe-area-context

用于图标

  • npm install @expo/vector-icons

这是一个很长的安装列表，但是我们必须安装它们，因为我们要在我们的应用程序中使用它们。
