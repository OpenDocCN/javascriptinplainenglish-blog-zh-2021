关于Vue.js 3支持静态站点生成器的状态

根据 Awesome Vue ，Vue.js有相当多的静态网站生成器可供使用，它们是 Gridsome 、 徐伟贤 、 Saber 、 Servue 、Vuers、T25(按字母顺序)。此外还有 Nuxt 3 (Nuxt.js v3)。

Tl；dr

在编写本报告时， Vue 3 的“转到”静态站点生成器是 VuePress v2 。一旦 Gridsome 支持 Vue 3 和 Nuxt 3 作为稳定版本上市，这两个都是其他合理的解决方案。

Nuxt 3

nuxt . js(git hub项目)是一个主要的 Vue.js 框架，支持静态站点生成。一段时间以来，有 Nuxt.js v3 又名Nuxt 3(git hub项目)可供测试版使用。

Gridsome

grid some(GitHub项目)是以 Netlify 和 algolia 为后盾的Vue.js的Jamstack框架。

Netlify 是为web应用和静态JAM stack网站提供托管和无服务器后端服务的主要提供商之一。 algolia 提供托管搜索API ( algolia搜索 )和托管推荐API ( algolia推荐 )。与其他可用的静态网站生成器相比，就赞助量和长期支持而言， Gridsome 的支持可以被认为是最“稳定”的。

Gridsome 可以看作是 React 静态网站生成器 盖茨比 的主要竞争对手。顺便说一下:在有盖茨比云 之前，Netlify也是盖茨比云的主要托管解决方案。

贡献频率不如VuePress，但还可以。大部分贡献来自Gridsome 的联合创始人Hans-Jr gen ved vik。

Gridsome还不支持Vue 3。有关更多信息，请参考GitHub上用于Vue 3升级的看板。

徐伟贤

徐伟贤 已存档停止开发。它的继任者是军刀。

军刀

Saber(GitHub项目)是徐伟贤的继任者。根据撰写本文时的发布页面(2021年8月8日)，最后一次发布saber@0.11.7是在2020年5月19日，这意味着很久以前。此外，从2020年10月起，投稿频率几乎降至零。

在我看来，项目也可以被认为是死了。

服务

Servue尚未发布一个正式版本并且自2020年10月以来没有显著的贡献。

在我看来，这个项目也可以被认为是死了。

VuePress

Vue press(Vue press v1)(GitHub项目)是知名的Vue静态站点生成器。对Vue 3的支持可通过Vue press-next(Vue press v2)获得。自2020年12月24日起，不再积极维护VuePress 1。这意味着所有的开发工作都投入到vuepress-next (VuePress v2)中。

对了，主要贡献者meteorloxy来自Vue开发团队。
