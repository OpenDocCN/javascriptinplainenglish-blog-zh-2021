*   ***例如，当 **attr** 被包裹在 **String** 对象中时*** 我们尝试访问 **String 对象**属性。

## 6.命令式编程和声明式编程有什么不同？

*   **命令式**编程主要关注如何控制代码执行的路径。
*   **声明性的**编程专注于做什么，这在构建**API**方面有优势。

## 7.陈述和表达有什么不同？

*   **语句**执行一个动作，其中**表达式**引出一个值的求值。
