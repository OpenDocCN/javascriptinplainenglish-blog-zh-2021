5。vue-prlx

Vue-Prlx是一个 Vue.js 视差指令，它将可定制的视差滚动效果应用到应用程序中的图像，它可以动画平移和背景位置。

  • 可定制的
  • 强烈推荐个人项目。

安装:

npm i vue-prlx

6。vue-典型

这是一个非常简单小巧的Vue.js组件，用于Vue.js应用程序的动画输入，也用于JavaScript 的 ~400字节的动画输入。由于这个库是相当新的，并且每个月都有更新的版本发布，所以建议只使用这个库来尝试您个人项目中的新内容。

  • 轻量级选手

安装:

NPM I vue-典型
