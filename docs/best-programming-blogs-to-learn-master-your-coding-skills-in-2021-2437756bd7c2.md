我们生活在程序的世界里，从智能手机上的打盹闹钟到在社交媒体上与朋友交谈后帮助我们入睡的应用程序。随着这个领域的蓬勃发展，成吨的资源正变得对我们可用，我们开始这场竞赛只是时间问题。

对于有经验的程序员来说，与世界保持同步总是更好的，因为语言和框架正在发生很多变化。

> 持续学习是任何领域成功的最低要求。布莱恩·崔西

这里有一些编程博客，可以帮助你启动你的编程生涯，并让你了解 2021 年的最新趋势。
