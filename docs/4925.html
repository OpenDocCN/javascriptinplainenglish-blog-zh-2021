在本文中，我将向您展示如何在不到5分钟的时间内部署一个 React 站点。以及如何在网上免费部署React网站。

如何在5分钟内部署一个React站点？

我将使用 GitHub 和 Netlify 使之成为可能。

为了在5分钟内部署React站点，我假设您已经有了一个工作的React站点，并且您的代码被推送到GitHub存储库。如果没有，请注册 GitHub ，创建一个新的库，并将您的工作React站点代码推送到新创建的GitHub库。然后再回到这篇文章。

别忘了启动你的秒表(设置5分钟倒计时)。

我们开始吧

对于本文，我将使用我个人的React GitHub存储库在Netlify上进行部署。以下是我的存储库的链接:

https://github . com/sayyedhammadali/Deploy-React-Site-On-net lify-Free

第一步:

到 Netlify 网站和 注册 创建一个账户。我通过GitHub注册是为了让事情变得更快。

第二步:

输入您的GitHub凭证，然后点击授权网络生活。

第三步:

现在，在Netlify上成功创建帐户后，我们将从Git 中点击新站点。

第四步:

选择 GitHub 上传你的React站点代码。正如我在文章开头提到的，我们需要在GitHub上有工作的React代码。您也可以选择其他选项，如GitLab或Bitbucket，但对于本文，我使用的是GitHub。

第五步:

现在点击授权Netlify 让Netlify访问GitHub代码库。您可以选择选项来访问特定的存储库或所有的GitHub存储库。

第六步:

如果您在列表中看到所有的存储库，那么选择您想要部署的存储库。在我的例子中，我将选择我的React代码库，这是我在本文前面提到的。

注意:如果你没有看到所有的库，通过点击链接在GitHub上配置Netlify app来配置GitHub显示所有的库(看下图)。

确保选项 All repositories 被选中并且被保存(看下图)。

斗牛中的最后一剑

现在，您可以在Netlify上看到React现场部署，不到5分钟完全免费。我希望你的秒表还有足够的时间。单击部署站点(其他设置保持不变，但可以根据您的需要进行更改)，然后等待Netlify部署过程完成。可能需要几分钟。我没有将这个时间包括在我们的React站点部署时间中。

搞定！请现在停止你的秒表。

等等… Netlify正在部署你的React站点，耐心点。这就是正在进行的Netlify部署过程中您的屏幕应该显示的样子。

单击自动生成的Netilfy站点链接访问您的站点。我的情况是:https://boring-tesla-b30e0f.netlify.app/
