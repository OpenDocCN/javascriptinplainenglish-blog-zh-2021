SQL 代表 S 结构化 Q 查询 L 语言。

发音为“Sequel”或“S-Q-L”。好玩吧？我们喜欢选择。特别是，我们发现我们使用SQL来操作和访问数据库，尤其是关系数据库。像SQL这样的查询语言是一种旨在帮助我们从数据库中检索特定数据的语言。有相当多的数据查询语言，但是SQL是开始学习数据查询的一个很好的切入点，也是我们现在要关注的。

SQL的诞生

SQL的一个更基本的版本是由两位IBM开发人员Donald D. Chamberlin和Raymond F. Boyce在20世纪70年代首先开发的，他们将SQL开发为一种在IBM的准关系数据库中操作和存储数据的方法。1986年到了，这是SQL成为美国国家标准协会(ANSI)确定的技术行业数据查询的技术标准的时候。
