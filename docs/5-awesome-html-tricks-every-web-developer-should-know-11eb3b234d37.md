# 1.延迟加载图像

你听说过懒装路线或者组件吗？图像也是如此。它们也可以是懒惰装载的。这将防止在屏幕上立即加载所有你并不真正需要的图像，从而加快你的应用程序的加载时间。由于它每次显示的图像数量，这也用在介质上！

