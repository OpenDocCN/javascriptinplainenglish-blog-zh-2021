让我们从使用下面的命令安装expo开始。

npm install -g expo-cli

之后，我们可以通过在终端中执行以下命令来初始化我们的项目

expo init myTodoListApp

我们的项目设置已经完成，所以我们可以继续前进，建立我们的TodoList应用程序。
