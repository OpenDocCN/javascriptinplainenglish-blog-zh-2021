在 style.css 内部

```
.columns2 p{column-count: 3;text-align: justify;column-gap: 100px;column-rule: dashed blue 10px;}
```

Column-count 是用来设置列数的，我们还想添加 text-align 来调整以拉伸每行文本的宽度。我们还将列间距设置为 100 像素。在列之间添加一条 10px 的蓝色虚线。
