我们将看到如何在页面加载时创建一个动画加载器。

基本上，它相当于JavaScript的load事件。当加载了整个页面，包括所有依赖的资源，如样式表和图像时，会触发load事件。

window.addEventListener('load', (event) => {
  console.log('page is fully loaded');
});

我们需要在public目录中的index.html文件中添加我们的加载器 HTML和CSS。

由于反应应用程序安装在根 div中，所以我们需要在根 div中添加我们的加载器HTML部分。

然后，我们可以将CSS部分添加到同一个文件中，在<style>标签中。

就这样，它将为您的React应用程序创建一个页面加载器，只有当网站首次打开时才会出现。

它与我们在React.lazy和Suspense情况下使用的非常不同，因为它使用了Suspense的fallback属性，该属性在我们页面的路线每次改变时都可见，并且是一个新的路线。
