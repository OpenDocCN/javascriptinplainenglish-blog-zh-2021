原文：https://javascript.plainenglish.io/how-to-create-a-youtube-downloader-with-node-js-and-react-a86d7586fcc8?source=collection_archive---------1-----------------------#2021-07-05

在本教程中，我们将构建一个YouTube下载器，后端在Express中实现，前端在React中实现。

app的基本流程:

  1. 用户将提供一个YouTube视频链接
  2. 后端服务器将在队列中推送该视频链接以处理下载
  3. 当作业从队列中弹出进行处理时，后端为客户机发出事件
  4. 客户端监听事件并显示适当的消息
  5. 用户可以从服务器上下载视频

我们将使用 Socket.io 进行发射事件和处理作业，我们将使用 Bull 包。

REST API

REST APIs Routes
1. GET => /api/donwloads => Get all downloads
2. GET => /api/donwloads/:id => Get a single download
3. POST => /api/downloads => Push new download
4. DELETE => /api/downloads/:id => Remove a single download
5. GET => /api/downloads/:id/downloadfile => Download a single file
