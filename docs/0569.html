使用Quasar库开发Vue应用程序—日期选择器选项

原文：https://javascript.plainenglish.io/developing-vue-apps-with-the-quasar-library-date-picker-options-60679182f663?source=collection_archive---------11-----------------------#2021-02-03

Quasar是一个流行的Vue UI库，用于开发好看的Vue应用程序。

在本文中，我们将了解如何使用Quasar UI库创建Vue应用程序。

默认日期选取器日期

我们可以用default-year-month属性设置默认的日期选择器日期:

<!DOCTYPE html>
<html>
  <head>
    <link
      href="https://fonts.googleapis.com/css?family=Roboto:100,300,400,500,700,900|Material+Icons"
      rel="stylesheet"
      type="text/css"
    />
    <link
      href="https://cdn.jsdelivr.net/npm/quasar@1.12.13/dist/quasar.min.css"
      rel="stylesheet"
      type="text/css"
    />
  </head>
  <body class="body--dark">
    <script src="https://cdn.jsdelivr.net/npm/vue@^2.0.0/dist/vue.min.js"></script>
    <script src="https://cdn.jsdelivr.net/npm/quasar@1.12.13/dist/quasar.umd.min.js"></script>
    <div id="q-app">
      <q-layout
        view="lHh Lpr lFf"
        container
        style="height: 100vh;"
        class="shadow-2 rounded-borders"
      >
        <div class="q-pa-md">
          <q-date v-model="date" default-year-month="2020/08"> </q-date>
        </div>
      </q-layout>
    </div>
    <script>
      new Vue({
        el: "#q-app",
        data: {
          date: undefined
        }
      });
    </script>
  </body>
</html>

当日期为null或undefined时，将显示默认日期。
