关于Vue 3支持的状态

最后更新:2022年1月21日

在撰写本文时，支持Vue 3的UI框架有:

  • 不可知论(GitHub上的26颗星星)
  • 放大UI(GitHub上的139颗星星)
  • Vue的Ant设计 GitHub 上的15.7k星
  • 元素加(GitHub上的10.2k星星)
  • 等于(GitHub上的667颗星星)
  • 原生UI(GitHub上的7k星星)
  • or uga UI(GitHub上的549颗星星)
  • PrimeVue (1.3k星星在 GitHub 上)
  • superb vue(GitHub上的11颗星)
  • 顺风UI (基于GitHub上带12k的无头UI )
  • vue stic(GitHub上的1k星星)
  • 类星体框架(GitHub上的18.7k恒星)

不可知性

AgnosticUI拥有对Vue3 的支持，并提供20个组件。

放大用户界面

Amplify UI 是一个组件库(38个组件)，具有对Vue 3 的支持，并专注于亚马逊AWS兼容性。这使得高级组件如聊天机器人或认证器成为可能。

面向Vue的蚂蚁设计

Ant Design for Vue(15.7k stars onGitHub)是知名React库Ant Design(“全球第二大流行React UI库”)的一个端口，提供60+组件。

BootstrapVue

BootstrapVue(13.2k stars onGitHub)支持在Vue 2项目中使用Bootstrap(151k stars onGitHub)组件。BootstrapVue 还不支持Vue 3。

布埃菲

Buefy(GitHub上的8.7k星星)将CSS框架布尔玛(GitHub上的43.7k星星)整合到Vue.js 2项目中。然而 Buefy还不支持vue . js 3。

核心用户界面

核心UI组件(GitHub上的346颗星)支持在Vue.js 2 projecs中使用以仪表盘为焦点的引导组件。核心UI管理模板(GitHub上的2.8k星星)提供了一个基于Bootstrap的管理模板。有vue . js3还没有稳定发布 。

元素加

Element Plus(10.2k stars onGitHub)是Vue.js 3的UI库。

平等的

Equal 是一个Vue 3组件库，拥有30+个组件，基于TypeScript和个人设计系统。

墨水线

Inkline(GitHub上的811颗星星)提供50+组件。Vue.js 3还没有稳定版本支持，但是 测试版应该支持Vue.js 3 。

本机用户界面

原生UI(GitHub上的7k星)是一个Vue 3组件库，提供了大约。80个组件。

奥鲁加UI

Oruga UI (在 GitHub 上549颗星)是一个25+ UI组件的轻量级库，没有CSS框架依赖，兼容vue . js 3。

iView

iView(GitHub上的23.8k星)不不支持Vue.js 3 。

Keen UI

Keen UI (星上 GitHub ) 难道还不支持Vue.js 3 吗。

薄荷UI

Mint UI(GitHub上的16.4k星星)不支持Vue.js 3 。

PrimeVue

PrimeVue (1.3k星星上 GitHub )提供90+组件，支持Vue.js 3。要将PrimeVue添加到Vue3项目，执行yarn add primevue@next，选择版本，例如3.9.1，并添加对带有yarn add primeicons的图标的可选支持。此外，您需要添加一些样板文件来包含组件(模块加载器设置变量)或直接导入组件(单个文件组件设置变量)。例如，为了使对话框组件<Dialog></Dialog>可用，必须向main.js添加以下代码:

...import PrimeVue from 'primevue/config';
//import Dialog from 'primevue/dialog';  // module loader variant
import Dialog from 'primevue/dialog/sfc';  // SFC variant...app.use(PrimeVue);
app.component('Dialog', Dialog);...

SuperBVue

superb Vue(GitHub上的11颗星)支持Vue 3和Bootstrap 5，并提供大约10个组件。

类星体框架

Quasar Framework(下) (18.7k stars on GitHub )是Vue.js 3的一个UI框架。

顺风UI

Tailwind UI 是一个UI组件库，与Vue.js 3 协同工作。它使用了引擎盖下的无头UI (12k on GitHub )。

使虚弱

vue tify(GitHub上的31.1k星)将符合材质设计规范的材质UI 组件使用到Vue.js项目中。作为v3 (Titan)版本的一部分，对复合API的全面支持将于2022年2月提供。

v3 alpha发布包可通过yarn add vuetify@next或npm i vuetify@next在下一个标签下获得。应该可以通过执行vue add vuetify将vuetify添加到Vue3项目中。

然而，预设选择

  • 默认(推荐)，
  • Vite预览(Vuetify 3 + Vite)，
  • 原型(快速开发)
  • 虚拟化3预览(虚拟化3)

执行yarn serve时导致错误Error: Cannot find module `fst/promises`或Error: You cannot call "get" on a collection with no paths. Instead, check the "length" property first to verify at least 1 path exists.。到目前为止，我还没有让阿尔法工作。

真空材料

VueMaterial (9.4k从 GitHub 开始)是Bootstrap到Vue.js项目的又一次集成。VueMaterial 尚不支持vue . js 3。

Vue材料套件

Vue材料套件 (239颗星在 GitHub ) 还不支持Vue . js 3。

VueTailwind

VueTailwinds (1.4k星在 GitHub )将CSS fram worktail wind CSS(42.8k星在 GitHub )集成到Vue.js项目中。我不确定它是否支持使用 Tailwind UI (专有，无限制使用从129欧元开始)组件。VueTailwind 尚不支持vue . js 3。

Vuesax

vue sax(git hub上的918颗星)处于alpha状态 还不支持vue . js 3。

vu esse

Vuest(在 GitHub 上用1k颗星表示)是一个包含50多个组件的Vue.js 3.0用户界面框架。

波形用户界面

Wave UI (310星在 GitHub )是一个包含40多个组件的UI组件库。
