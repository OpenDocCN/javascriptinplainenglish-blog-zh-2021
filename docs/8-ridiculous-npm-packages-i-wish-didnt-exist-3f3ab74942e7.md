# 6.是奇数

这个软件包消除了你检查一个数字是否是奇数的麻烦。

![](img/7ec18a977e203a2383f833b2ee84e33e.png)

is-odd

这个包有**百万**的下载量，这个包有超过 **20 位作者**和 **6 位作者**来写这个。
