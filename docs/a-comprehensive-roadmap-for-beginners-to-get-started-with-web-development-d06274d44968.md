# 学习 Web 开发

嗯，我学过编程语言、数据结构、web 开发、一点机器学习和数据科学的概念。

除此之外，我认为学习 Web 开发很有趣。当你开始使用 HTML、CSS 和 JavaScript 时，你完全可以创建一个作品集网站，而不需要 React、Angular、Vue、MongoDB、Strapi、TailwindCSS、Bootstrap 和许多其他类似的技术。
