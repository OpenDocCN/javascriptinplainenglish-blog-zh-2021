**01。** **错误处理:**有时候我们的代码会有错误。这些错误的发生是因为我们的错误，可能是一个意外的输入，或者是一百个其他的原因。所以，我们使用 **try…catch** 来处理这些错误。

**try…catch:** 它有两个主要模块: **try** 和 **catch** 。首先执行“try”块。如果没有错误，则**捕捉(错误)**被忽略。如果有发生，那么 **try** 执行停止，然后 catch(error)显示错误。
