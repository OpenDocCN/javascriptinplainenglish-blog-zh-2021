2.null和undefined有什么区别？

答:首先， null 表示空的或者不存在的值。应该是不存在的东西。它可以作为空值的表示形式赋给变量。

另一方面， undefined 意味着一个变量已经被声明，但是它的值还没有被赋值。在函数中，如果我们不传递参数，就会显示undefined。

3.两倍等于(==)和三倍等于(===)有什么区别？

回答:双等号==是一个抽象的等号运算符。它通过将表达式从一种数据类型更改为另一种数据类型来检查其两个操作数是否相同。

另一方面，Triple equal ===是一个严格的等式运算符，对于不同类型的值，它返回false。三重等于不执行类型强制。

8。Let和Const有什么区别？

回答:在Es6版本的JavaScript中，变量可以多两种方式声明。首先，我们可以用关键字 let 声明一个变量。

通过应用关键字 let ，我们可以不止一次地改变变量的值。如果我们在一个块范围内使用关键字 let ，那么我们将不会在这个块范围之外访问它。我们也可以通过应用关键字 const 来声明一个变量。但是它的特点是变量值不能改变多次。
