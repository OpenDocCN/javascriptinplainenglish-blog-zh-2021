我们正在提升我们的生活方式、场所和家园。网站也不例外，甚至他们正在升级到一个新的水平。20 年前，人们对 React、Angular 或 Vue 并不狂热。没有像 Bootstrap、Material-UI、Tailwind 这样的 CSS 框架。

以前，每个人都是在没有模型的情况下工作的。但是现在一个 web 开发人员必须学习很多东西，管理很多东西，编写很多代码。

**注意:**我的导师给了我所有这些工具的建议，所以我想与网络世界分享它们会很好。我不隶属于这些工具的制造商，因为他们可以自由使用(在一定限度内)。

所以，为了学习新的概念，我和一些网络开发专家进行了交流。其中，我还和一位最好的导师交谈过，他正在一家顶级国际公司工作和管理。他每周都打电话来帮助我获得洞察力(实际上我正在帮助他管理一些任务，所以他帮助我获得这种洞察力)。

他是一名经验丰富的 web 开发人员，拥有超过 30 年的工作经验。我还向他询问了初学者取得成功的最佳工具。

下面是他的回答:“正如你所知，我学习和实现 web 开发已经超过 30 年了。在我职业生涯的最初阶段，没有任何这样的工具。我们(开发人员)并不精通技术。随着时间的推移，新的工具被发现，我们用它们来取得成功，变得更有生产力，赚得更多。”

第一个是特雷罗。

特雷罗:首先，也是最重要的一点，你必须管理好自己的时间。为此，特雷罗出现了。Trello 是一个简单且免费的网站(也有一个应用程序)。你可以管理那里的一切，你甚至可以与每个人分享。

我每天晚上都用它来写下第二天的任务，并为要做的工作、正在回顾的工作、已完成的任务等等添加卡片。

是的，如果我在一个团队工作，我可以分享给我的队友和其他人。

我在家里开始了我的网页开发工作。我曾经作为一名 solopreneur 创建网站和设计网站，使用 Photoshop，实体模型，前端，后端，甚至管理数据库都是乏味的工作。

但是现在，相信我，你不能从零开始。一段时间后，你的大脑讨厌同样的工作。所以我参考了 [Dribbble](https://dribbble.com/) 的说法。它帮助我想象世界上正在发生的事情，甚至帮助我为我的下一个网站和应用程序设计找到一个想法。

**3) Figma:** 两个几十年前，我们一般会和一个客户谈他们的网站，然后开始编码。而且当时一个客户也没有为他们的网站设计辩解。但是今天它取决于客户是接受还是拒绝一个设计。

现在，如果你开始你的 web 开发项目是直接跳进编码，那么你就是最没脑子的人。首先，你必须倾听你的客户，然后努力创造一个设计。我用的是 Adobe XD 和 Figma。

但是你可以使用 Figma，因为它是免费的，而且是一个很好的工具。

网站的一个重要方面是看起来有吸引力以吸引顾客。SVG 扮演着重要的角色。

![](img/d96da26a71fd8db97faae71a47910630.png)

Source: Canva

SVG(可缩放矢量图形)只是一个允许我们在网站上显示矢量图像的代码。

更清楚地说，正如可缩放矢量图形的名字所暗示的那样，它是:

1.  **可缩放**:您可以缩放您的图像而不会损失质量，因为它不是像素化的。
2.  **矢量**:光栅图像由分辨率看起来很怪异的像素组成。为了解决这个问题，矢量图像应运而生。简单的矢量图像是由数学公式构建的。例如，Photoshop 基于像素，而 Illustrator 基于矢量。
3.  图形:图形是一个物体的图像、设计或视觉表现。

因此 SVG 由三个词组成——可伸缩、矢量和图形。

现在您已经了解了 SVG，您可以创建它，但是这会耗费您的时间。所以，试着用[拔](https://undraw.co/illustrations)来完成。[这里](/how-to-create-an-svg-animation-illustration-for-your-website-d5a31ba2e712)是使用它的简要指南。

Canva:

如果你问我你最爱什么，我的答案肯定是 [Canva](https://www.canva.com/) 。人们通过制作脸书广告、制作视频、平面设计等等来赚钱——显然是免费的。

让我告诉你一个有趣的故事。我曾和一家公司合作，有一个女孩是平面设计师，制作模板、演示文稿和所有那些疯狂的东西。老板真的很欣赏她，但她实际上每天工作不到 3 个小时。

我真的被这震惊了，所以我问她是怎么做到的。她告诉我她在用 Canva 做这个。
