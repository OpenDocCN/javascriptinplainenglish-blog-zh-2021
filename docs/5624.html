初始化您的项目

让我们首先初始化一个空的Node.js项目。转到您的终端，运行以下命令

mkdir web-scrapercd web-scrapernpm init -y

这将为你创建一个package.json文件。

然后创建我们的根文件

touch index.js

所以我们现在有了一个skeleton Node.js项目。让我们使用它！

安装依赖项

运行以下命令获取所需的依赖项

npm i axios cheerio

打开您的index.js文件并导入它们

const axios = require("axios");const cheerio = require("cheerio");
