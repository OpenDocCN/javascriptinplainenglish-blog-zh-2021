如何将React应用程序部署到Netlify

在几分钟内部署您的React应用

React是用于前端开发的最流行的开源JavaScript库之一。脸书在2013年发布了这个项目。React允许我们使用可重用的UI组件，使得维护和扩展web应用程序变得更加容易。

在这篇文章中，我将向您展示如何将React应用程序部署到 Netlify 。Netlify是一个允许我们从Git构建和部署站点的平台。此外，它还带有其他很酷的功能，如自定义域、HTTPS、部署预览、回滚，最重要的是，还有一个 免费 层！

设置

对于这个帖子，我将使用 GitHub 和Netlify。首先，我们需要创建一个git存储库，这样我们就可以用Netlify连接它。

你可以从我为这篇文章创建的GitHub 下载git资源库。现在，让我们从Git 按钮点击 新站点，转到 Netlify 创建一个新站点。

现在，选择您更喜欢的Git提供者。就我而言，我选择了GitHub。如果您看不到存储库，请单击在GitHib上配置Netlify应用程序链接。

选择存储库后，确保构建命令和发布目录字段类似于以下值。

最后，点击部署站点按钮。部署站点需要几分钟时间。部署后，您可以通过Netlify生成的URL访问它。就我而言，我的网址是https://thirsty-stonebraker-77b897.netlify.app/。

因为我们已经将git repo连接到Netlify，所以对主分支的任何更改都将触发生产部署。
