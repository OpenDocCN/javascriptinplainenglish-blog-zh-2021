你是这个等式中不可或缺的一部分，你选择的资源也起着重要的作用。

不可否认，你是那个把努力放在学习 JavaScript 上的人。几个小时是你几乎每天应该带的最少时间，但你不是唯一一个出现并做好工作的人。你所选择的资源的创造者和你一样有义务把他们最好的发挥出来。

我做了一个[少数脸书](https://www.facebook.com/groups/javascript.react.node/permalink/1516921615338869/) [民意调查](https://www.facebook.com/groups/1605550896386197/permalink/3089967281277877/)，从中我可以提炼出为什么几乎每个人都需要 6-12 个月的三个细节，并找到一个[关于如何将这个过程加快到需要 2-3 个月的激动人心的 quora 答案](https://www.quora.com/How-long-did-it-take-you-to-learn-JavaScript)。

## 1.选择适合您的资源

免费资源看似完美，但价格却隐藏在众目睽睽之下。

某样东西免费是有正当理由的，它只是一个大集合中的一部分。你需要在网上挖掘，把所有的碎片收集在一起，我以为你想成为一名开发者，而不是收藏家。

想象接近一个全新的主题，那么你应该如何知道你需要收集哪些作品呢？开始学 C#编程的时候，也是搜索免费内容，分不清自己需要什么，避免什么。没有人为我筛选资源，我一片一片地消耗，不知道哪里会结束。这条道路比语言本身更令人困惑。此外，试图猜测一块拼图的重要性是压倒性的。

由于我多年前犯了收集错误，我知道这个话题的重要性。这是关于正确的资源和创作者有多关心你和你的目标，而不是推出一个巨大的主题内容列表。Hazel Jones 和其他[人](https://www.quora.com/How-long-did-it-take-you-to-learn-JavaScript)说他们只需要 2-3 个月就能成为一名 JS 开发者。由于选择了正确的资源，她甚至在 Airbnb 找到了一份工作。

[我还创建了 JavaScript，HTML & CSS，C#，SQL (Youtube)](https://www.youtube.com/channel/UCfSQ2-Suwq_axrMc8DIxD6g) 视频课程。我知道你应该对内容进行多少检查，以及创作者是如何关心简单的解释和引人入胜的交付。

## 2.你需要几乎每天都致力于这个话题

学习 JavaScript 不是一件容易的事情，有很多东西需要学习。

这种语言为你提供了一大堆你可能从未听说过的新单词、概念和奇怪的东西。除了日常生活之外，没有什么是你能轻易做到的。

计算机科学学位的存在是有原因的。我成为一名全栈式网络自由职业者的旅程也很漫长，但由于这项技能为我提供了机会，我每天在完成日常学习任务后都会全身心地投入其中。在学习期间，我接触到了编码，并花了几个晚上来构建我的项目，以深入挖掘并尽可能多地了解。有很多事情要处理，也有很多东西要学。当你把它拿到手的时候，这个列表只会变得更小。

学习一门编程语言意味着展示和保持一致。在一周中有固定的时间框架建立了一个惯例，并且习惯这个惯例也帮助我变得成功。与你的工资/贷款相比，你只有在工作时才能得到报酬。

虽然你选择的资源起着重要的作用，但你需要出现并解决抛给你的话题。

## 3.认真思考你为什么想学 JavaScript

[一些人回答我的民意调查说，他们对 10 多年的工作经验没有信心。](https://www.facebook.com/groups/1605550896386197/posts/3089967281277877/?comment_id=3090014987939773)

我想知道这些年来他们都做了些什么，我想他们没有合适的理由开始开发工作。你看到了让自己深入思考为什么要成为一名 JavaScript 开发人员的重要性。

不管原因是什么，它必须服务于你，而不是你生命中的其他人。对我来说，我之所以还在这个行业，尽管每天都是狗屎和挣扎(即使是 7 年后)，是因为我可以在我统治的地方创造我的帝国。人们听了我的话，当我说“T4，这是不对的”时，他们改变了计划。我们应该也必须以相反的方式去做。”因此，找到你自己的理由对战胜这一挑战至关重要。艰难的日子会到来，学习语言是你自己的选择。

拥有坚定的信念/理由是这些时候的燃料。

对你来说重要的一个原因是基础，它决定了你成为 JavaScript 开发者的速度。也许你也听过这样的说法:*“嗯，我开了 20 年车，所以相信我说的话..BLA BLA"* ，我对这句话的想法是:*“是的，你可能有超过 20 年的执照，但你仍然可以像狗屎一样驾驶超过 20 年。这不是一个有效的论点。”*总结一下，就是你对题目的关心程度，以及你想达到什么目标。

慢慢来，这个决定可能会改变你的生活，对我来说，它是(我放弃了我有文凭的领域)，当它成为现实时，这是你的新生活，只有你必须过它，没有其他人。

**学习 JavaScript 或编程语言**，总的来说，不是一个简单的话题，也不是在短时间内完成的，你有足够的时间来做研究，找到适合你和你学习风格的资源([也许你可以看看我的](https://www.youtube.com/channel/UCfSQ2-Suwq_axrMc8DIxD6g)？).

**即使你已经找到了合适的资源**，这也是关于你如何处理它们和创作者的任务，以保持它的吸引力和刺激性，这就是为什么我制作了一个 [SQIsLand-Pirate Adventure 来教授 SQL](https://cntr.click/jgT6N22) 。

**最后但同样重要的是**，确保你走上这条路是出于正确的原因，而且只适合你，不适合其他人。

不要拐弯抹角，坚持住。
