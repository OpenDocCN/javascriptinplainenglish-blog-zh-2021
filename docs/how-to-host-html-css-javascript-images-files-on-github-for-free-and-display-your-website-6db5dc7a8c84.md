## 1.创建 Github 帐户

去 github.com 创建一个免费帐户。你的账户会在几秒钟内准备好。

## 2.创建新的存储库

当您登录到您的帐户时，在屏幕左侧，单击“新建”。

## 3.为您的存储库命名

命名您的存储库。

如果你想让人们对它有所贡献，你可以把它公开。现在点击添加一个自述文件。勾选 README 文件后，您可以解释您的存储库是关于什么的。

然后单击创建存储库。

## 4.上传 HTML，CSS，图片文件

在新页面上，单击主框顶部的添加文件。将弹出两个选项—创建新文件和上传文件。

单击上传文件，从您的计算机或设备上传您的文件。

## 5.拖动或上传文件

您将看到拖动文件或从计算机上传文件的选项。你可以选择这两个选项中的任何一个。但是请注意。

你应该把所有的文件放在一个文件夹里——HTML、CSS、JavaScript 和图片文件。此外，给文件夹一个一个单词的名称(例如，网站)。将您的 HTML 文件命名为“index.html”。

这些对于让你的网站在 Github 上显示很重要。

等等，让文件上传。在你上传文件的方框下面打勾。你会知道你的文件是否上传。

上传完成后，单击提交更改。

## 6.转到设置

在您现在所在的屏幕上，您会看到设置—在右侧，在框的顶部。点击它。

现在向下滚动到 Github 页面。点击——点击此处查看。

它会引导你到新的 GitHub 页面。在“来源”下方，点击无。然后点击“主”。现在点击保存。

您将会看到这条消息—您的网站已准备好发布在……

几分钟后，点击链接。

一切正常，当你点击那个链接时，你的网站应该会显示，但它不会。

您只能看到您的存储库名称或您在自述文件中写的内容。甚至是 404 错误。以下是你如何解决这个问题，并使你的网站显示。

将您上传的文件夹名称和您的 HTML 文件名称添加到为您生成的链接 GitHub 页面的末尾。例如:

[https://ololade97.github.io/cak/ola/index.html](https://ololade97.github.io/cak/ola/index.html)(此链接为举例，非真实)

“ola”是我的文件夹的名称，“index.html 是我的 html 文件的名称。

嘣。您的 GitHub pages 网站已经建立并运行——看起来很漂亮。
